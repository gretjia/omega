终端中的仪表盘是 PROJECT OMEGA 的战术雷达，各个栏目的含义如下：

1. 基础信息栏
CODE: 股票代码（如 603228）。
ASSET: 股票名称（如 景旺电子）。
PRICE: 实时价格及当日涨跌幅。
绿色/红色: 对应涨跌状态。
2. 核心数学指标 (你的猎杀依据)
这是 M4 Max 计算的核心产物，也是你区别于散户的关键：

HAWKES (n): 爆发力 (Reflexivity)
定义: 基于 Hawkes 过程计算的“资金自激程度”。也就是“现在的每一笔买单，能引发多少笔后续买单”。
数学阈值: n > 0.85。
实战含义: 数值越高，代表资金越疯狂（Fomo情绪）。
操作: 看到它变红 (>0.85) 说明主升浪加速中，适合追涨（MOMENTUM）。
TDA (Life): 结构力 (Structural Stability)
定义: 基于拓扑数据分析 (TDA) 计算的价格形态“寿命”。
数学阈值: Life > 1.2。
实战含义: 数值越高，代表庄家的控盘结构越稳固，所有的波动都是噪音，核心趋势未变。
操作: 看到它变绿 (>1.2) 说明庄家完成吸筹并锁仓，适合买入持有（LOCK-IN）。如果数值掉到 0.3 以下，说明结构崩塌，庄家跑路（DECAY）。
3. ACTION (作战指令)
系统综合上述指标直接给出的操作建议：

SCANNING (灰色): 扫描中，无异常，空仓观望。
[LOCK-IN (BUY)] (红底白字): 结构锁死，买入。
触发条件：Life > 1.2 (结构极稳) 且 Entropy < 0.65 (熵减，有序)。
典型标的：景旺电子
