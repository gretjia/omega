# OMEGA Code Archive

This folder contains archived code, tools, and scripts from legacy versions (e.g. v62) of the Omega project.
These items are preserved for auditing and historical rollback purposes.
Data and runtime snapshots can also be found here.

- `audit/`: Archived audit reports and manifest checks (e.g., from v62 and older).
- `scripts/`: Archived shell and powershell scripts previously used for staging/data movement.
- `tools/`: Archived operational python tools.

For current operations, refer to the root `README.md` and the `handover/` directory.
